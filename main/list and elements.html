block elements

block elements are those elements which starts always from the new line or from next line

examples :

  1. h1-h6 tags
  2. p tags for paragraph
  3. list tags
  4. nav tag

inline elements

inline elements are those which starts from the same line. it do not change it's line

examples :

  • image tag
  • video tag
  • audio tag
